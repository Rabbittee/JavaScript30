## Day 2

### title 
模仿真正的時鐘。

### skill
1. 定時器使用的方式
2. querySelector() 選擇器的應用
3. transform-origin 指針的參考點座標
4. transform rotate 的應用
5. date prototype 的應用
6. 時針、分針、秒針的角度計算

### logic 
1. 獲取時針、分針、秒針的 dom
2. 取得當前時間的時、分、秒，時換算成 12 小時制以利後續計算
3. 時鐘原本誤差 90 度，調整時針、分針、秒針的指針位置
4. 計算時針、分針、秒針的角度
5. 每秒更新一次時間，讓時鐘動起來
