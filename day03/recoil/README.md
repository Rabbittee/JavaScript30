# Javascript 30 Days

## day03 - CSS Variables

#### 邏輯

1. 選取 .controls 下的 input
2. forEach input 的 value 去初始化 documentElement.style 的 css 變數
3. 監聽每個 input 的 change 與 mousemove 事件
4. 每次觸發事件就設定 css 變數
