### Skill
- canvas
  - 選取 canvas 畫布 (canvas.getContext('2d'))
  - ctx 變數
    1. width
    2. height
    3. strokeStyle
    4. lineCap
    5. lineJoin
    6. lineWidth
  - ctx function(?) 應用
    1. beginPath (開始)
    2. moveTo (線的起點)
    3. lineTo (線到哪裡)
    4. stroke (讓線出來)
  - hsl()：設定顏色(hue)、飽和度(saturation)以及光亮度(lightness)
